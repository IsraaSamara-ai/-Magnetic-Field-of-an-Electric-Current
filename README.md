Project Description
An interactive educational application designed to explain the magnetic field generated by an electric current in various conductors. The app features interactive 3D animations, a virtual laboratory, real-life examples, and a comprehensive final assessment. It serves as a learning aid for 12th-grade physics students.

Features
Biot-Savart Law - Interactive experiment with mathematical derivation
Magnetic Field from straight wires, circular coils, and solenoids with animated right-hand rule
Plasma Confinement by magnetic fields linked to nuclear fusion reactors
Force Between Parallel Conductors - Interactive experiment with mathematical derivation
Natural Magnetism - Magnetic domains and atomic magnetic moments
Charged Particle Motion in uniform magnetic fields (3D graphics)
Mass Spectrometer & Synchrotron with real-world applications
Force on Current-Carrying Conductor with Fleming's left-hand rule
Interactive Virtual Lab for measurement, observation, and conclusion
Interactive Examples with step-by-step solutions
Final Assessment - Interactive quiz (10 questions)
Technologies
